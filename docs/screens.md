# mentalcare - 画面設計

## 画面一覧

### 共通
| ID | 画面名 | 説明 |
|----|--------|------|
| S-01 | ログイン | Google OAuth / 匿名ログイン選択 |

### 相談者
| ID | 画面名 | 説明 |
|----|--------|------|
| U-01 | カウンセラー一覧 | オンライン中のカウンセラーを表示・選択 |
| U-02 | カウンセラー詳細 | プロフィール確認・セッション開始 |
| U-03 | セッション開始確認 | OK / Cancel ダイアログ |
| U-04 | セッション（チャット） | テキストチャット画面 |
| U-05 | セッション（音声） | 音声通話画面 |
| U-06 | セッション終了 | 終了確認・ホームへ戻る |

### カウンセラー
| ID | 画面名 | 説明 |
|----|--------|------|
| C-01 | プロフィール設定 | 名前・自己紹介の登録・編集 |
| C-02 | 予約管理 | 待機枠の作成・待機開始 |
| C-03 | セッションリクエスト | OK / Cancel の受諾ダイアログ |
| C-04 | セッション（チャット） | テキストチャット画面（相談者と共通） |
| C-05 | セッション（音声） | 音声通話画面（相談者と共通） |
| C-06 | セッション終了 | 終了確認 |

---

## 画面遷移

```
[S-01 ログイン]
    ├─ 相談者として入る
    │   └─ [U-01 カウンセラー一覧]
    │           ├─ カウンセラーを選ぶ → [U-02 カウンセラー詳細] → [U-03 確認] ─┐
    │           └─ 「今すぐ話す」────────────────────────────────── [U-03 確認] ─┤
    │                                                                             ↓
    │                                                              [U-04 チャット] or [U-05 音声]
    │                                                                             ↓
    │                                                                      [U-06 終了]
    │                                                                             ↓
    │                                                                    [U-01 一覧に戻る]
    │
    └─ カウンセラーとして入る
        └─ [C-01 プロフィール設定]（初回のみ）
                └─ [C-02 待機画面]
                        └─ リクエスト受信 → [C-03 受諾確認]
                                                ↓ OK
                                   [C-04 チャット] or [C-05 音声]
                                                ↓
                                          [C-06 終了]
                                                ↓
                                          [C-02 待機画面に戻る]
```

## ルーティング（Astro）

| パス | 画面 | 備考 |
|------|------|------|
| `/` | S-01 ログイン | 未ログイン時のトップ |
| `/counselors` | U-01 カウンセラー一覧 | 相談者ホーム |
| `/counselors/[id]` | U-02 カウンセラー詳細 | |
| `/session/[id]` | U-04 or U-05 セッション | チャット・音声共通 |
| `/counselor/[id]/appointment` | C-02 予約管理 | カウンセラーホーム |
| `/counselor/profile` | C-01 プロフィール設定 | |
