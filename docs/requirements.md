# mentalcare - 要件定義

## コンセプト

メンタルが病みやすい現代社会において、**すばやく・親身に**相談できる場を提供するメンタルカウンセリングマッチングアプリ。

---

## ユーザー種別

| 種別 | 説明 |
|------|------|
| 相談者 | メンタルの悩みを相談したい一般ユーザー |
| カウンセラー | 相談を受ける側（資格不問） |

---

## コアバリュー

- **すばやく**: 病んでいる人がすぐに相談相手を見つけられる
- **親身に**: 寄り添えるカウンセリング体験

---

## KPI

| フェーズ | KPI | 説明 | 目標 |
|---------|-----|------|------|
| MVP | マッチングまでの時間 | 相談したいと思ってから会話が始まるまで | 5分以内 |
| MVP後 | セッション満足度 | セッション終了後の5段階評価（評価機能実装後） | 平均4以上 |

---

## 法的・規制上の確認事項

- 日本では「カウンセラー」は業務独占資格ではなく、無資格でも活動可能
- 「傾聴・話を聴く」行為自体は法律で禁止されていない
- **禁止される行為**: 診断・投薬（医師法により医師のみ）
- プラットフォームとして「医療行為ではない」と利用規約に明示する必要あり
- ※品質管理・緊急時フローはMVP後に検討

---

## マッチング仕様

### マッチング方式（2種類）

| 方式 | 説明 | 向いている人 |
|------|------|-------------|
| 指名選択 | オンライン中のカウンセラー一覧から自分で選ぶ | 誰に話すか選びたい |
| 即時接続 | 「今すぐ話す」で空いている人に自動接続 | とにかく早く話したい |

### カウンセラー一覧の表示順
- オンライン中のみ表示（オフラインは非表示）
- 表示順はランダム or 接続開始時間順（MVP）

---

## MVP スコープ

### 含めるもの

**相談者側**
- ログイン（匿名 or Google OAuth）
- オンライン中のカウンセラー一覧を見る
- 指名してセッション開始
- 「今すぐ話す」で自動マッチング
- セッション：音声 or チャット

**カウンセラー側**
- ログイン・プロフィール登録（名前・自己紹介など）
- オンライン / オフライン切り替え
- セッションを受けて音声 or チャットで対応

### MVP後に検討
- **評価システム**（双方向・非対称表示）
  - 相談者 → カウンセラーを評価（公開）
  - カウンセラー → 相談者を評価（カウンセラーのみ閲覧）
- カウンセラーの審査・品質管理
- 緊急時フロー（自殺念慮など）
- 絞り込み検索・フィルター（専門分野・言語など）
- 料金・決済機能
- ビデオ通話
- カウンセラーが忙しい・無応答時の自動振り替え

---

## ユーザーフロー

### 相談者

```
ログイン
  ├─ カウンセラー一覧を見る → カウンセラーを選ぶ ─┐
  └─ 「今すぐ話す」────────────────────────────┘
                                                ↓
                                         OK / Cancel
                                                ↓ OK
                                    チャット or 音声セッション
                                                ↓
                                          セッション終了
```

### カウンセラー

```
ログイン → プロフィール設定 → オンラインにする
                                    ↓
                            セッションリクエスト受信
                                    ↓
                               OK / Cancel
                                    ↓ OK
                            チャット or 音声セッション
                                    ↓
                                セッション終了
```

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | Astro + React islands |
| バックエンド | SST + Lambda (TypeScript / Node.js) |
| 音声通話 | Amazon Chime SDK (WebRTC) |
| リアルタイムチャット | API Gateway WebSockets |
| 認証 | Google OAuth / 匿名ログイン |
| DB | DynamoDB |
